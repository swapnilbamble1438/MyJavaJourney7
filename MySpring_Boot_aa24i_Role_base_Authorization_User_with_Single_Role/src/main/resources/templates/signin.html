Role Base Authorization Application
In this Application there are four Roles:
1)USER : he can only view the data
2)CREATOR : he can only add new data
3)EDITOR : he can only edit existing data
4)ADMIN : he can view,add,edit,delete the data

In this application
thymeleaf-extras-springsecurity5 dependency
