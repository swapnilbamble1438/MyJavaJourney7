Role Base Authorization Application

In this Application there are four Roles:

USER: can only View the data
CREATOR: can only Add new data
EDITOR: can only Edit existing data
ADMIN: can View, Add, Edit, Delete the data

In this application
thymeleaf-extras-springsecurity5 dependency
has been used.
